# Что нового?

---

| Что изменилось? | Что это даёт? |
|---|---|
| Блок «Инструменты» в настройках Downloader получил новый стеклянный дизайн | Встроенная карточка статуса теперь повторяет основной визуальный стиль: показывает готовность зависимостей, версии `yt-dlp`, `ffmpeg` и `Deno`, путь к папке инструментов и компактное меню дополнительных действий |
| Для YouTube во вкладке «Загрузчик» добавлены video-preview и live preview | Под интерфейсом вкладки по-прежнему работает приглушённый фоновый preview, а по кнопке теперь можно открыть отдельное модальное окно с live player и звуком; при обновлении preview-source он старается продолжить и звук, и позицию того же ролика |
| Статус инструментов Downloader перенесён в footer верхнего состояния | Блок с `yt-dlp`, `ffmpeg` и `Deno` теперь живёт рядом с обозначением текущего раздела в нижней панели, показывается только у верхней границы страницы и по-прежнему отключается из Настроек |
| Глобальный progress bar перенесён в кнопку вкладки «Загрузчик» | Прогресс загрузки теперь остаётся рядом с навигацией и одинаково виден и в header, и в footer без отдельной верхней полоски |
| Переработан фон основных страниц и настроек | Downloader, Tools, Backup и Settings получили разные атмосферные фоновые режимы, которые сохраняют тему приложения, но лучше разделяют экраны визуально |
| Исправлено поведение опций разворачивания окна на Windows | Настройки разворачивания при копировании URL и после завершения загрузки теперь корректно максимизируют уже открытое или свёрнутое окно, не поднимая его из скрытого состояния в трее |
| Исправлено зависание tooltip после закрытия настроек | Footer-кнопка настроек больше не оставляет видимый tooltip после открытия и закрытия модального окна |
| Исправлен текст сетевой ошибки для превью и загрузок | Таймауты при обращении к не-YouTube источникам теперь показывают нейтральное сообщение про источник, без ошибочного упоминания YouTube |
| Кнопка «История» перенесена из шапки в footer | Она находится рядом с настройками в нижней панели и показывается только во вкладке «Загрузчик», не перегружая верхнюю строку быстрых действий |
| Из верхней панели убраны Настройки и Смена темы | Верхнее меню стало чище и оставляет только быстрые действия, а настройки и возврат наверх остались в footer |
| Навигация между вкладками теперь мигрирует из шапки в footer | В верхней части экрана вкладки остаются в header, а при прокрутке вниз автоматически переезжают в нижнюю панель вместе с кнопкой быстрого возврата наверх |
| Интерфейс обновления приложения очищен и улучшен | Обновления теперь используют один flyover-сценарий у версии приложения, показывают состояние проверки и актуальной версии, сохраняют компактный статус «готово к перезапуску» и различают сетевые, загрузочные и установочные ошибки |
| Добавлена новая цветовая тема «Изумруд» | В приложении появился ещё один спокойный тёмный вариант с зелёно-бирюзовыми акцентами, доступный и в настройках, и в окне первого запуска |
| Режим разработчика теперь сохраняется и управляет UI шире | После перезапуска и смены языка режим не сбрасывается, в нём можно отключить вкладку «Загрузчик», а верхняя панель скрывает лишние быстрые действия |
