# Что нового?

---

| Что изменилось? | Что это даёт? |
|---|---|
| Переработана вкладка «Внешний вид» в настройках | Параметры языка, темы, размера текста и режима экономии эффектов теперь сгруппированы понятнее и выглядят согласованнее с остальными вкладками |
| Упрощён встроенный блок «Инструменты» во вкладке «Загрузчик» | Интерфейс стал компактнее, тише и логичнее: дублирующие действия убраны, а основные статусы и путь к папке считываются быстрее |
| Вкладка «Загрузчик» в настройках приведена к единому виду | Карточки, внутренние группы и блоки параметров теперь визуально согласованы с остальными вкладками настроек |
| Блок «Инструменты» во вкладке «Загрузчик» встроен прямо в настройки | Проверка статуса, действия и папка инструментов теперь доступны сразу в текущем окне без отдельной модалки |
| Окно настроек переведено в более flat-стиль | Интерфейс стал тише и компактнее: меньше тяжёлых эффектов, меньше визуального шума и более предсказуемая структура секций |
| Разделы настроек уплотнены | Убраны лишние декоративные блоки, уменьшены отступы и минимальные высоты, поэтому разделы реже упираются в скролл |
| Переработан интерфейс инструмента «Проверка хеша» | Основной сценарий проверки стал понятнее, а сравнение со вторым файлом теперь раскрывается только когда оно действительно нужно |
| В проверке хеша появилась drop-зона для файлов | Можно перетаскивать один или два файла прямо в окно приложения и сразу запускать проверку или сравнение |
| Модальное окно настроек теперь открывается по центру окна приложения | Настройки выглядят стабильнее и визуально не прилипают к верхней части интерфейса |
| Сокращён шум в стартовых логах приложения | При запуске проще заметить действительно важные события, а служебные подробности больше не забивают журнал |
| Кнопка открытия папки настроек теперь открывает саму папку без выделения `wireguard.conf` | Навигация в настройках стала предсказуемее и не выглядит как открытие конкретного файла |
| Убрано дублирование логов при восстановлении папки загрузки на старте | В логах запуска осталось меньше шума, при этом порядок инициализации не изменился |
| Исправлены проблемы со скроллом и окном настроек | Приложение надёжнее восстанавливает прокрутку после потери фокуса, а модалка настроек больше не уезжает за пределы экрана на длинных разделах |
| Переработан интерфейс File Sorter | Верхняя панель, правила и preview-область стали контрастнее, чище и удобнее для чтения результатов сортировки |
| Уплотнён визуал File Sorter | Верхние controls, настройки и preview теперь занимают меньше места и быстрее считываются на одном экране |
