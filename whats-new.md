### Интерфейс

- **Упрощен блок WG Unlock**: убрана кнопка «Зачем нужна эта вкладка», чтобы в быстрых действиях остались только рабочие инструменты.
- **Улучшен предпросмотр dry-run в File Sorter**: теперь перед запуском в режиме предпросмотра показывается наглядная карточка с метриками и списком первых действий сортировки.
- **Улучшен блок выбора папки в File Sorter**: кнопка «Открыть папку» теперь находится справа от отображаемого пути, а на мобильных элементы выстраиваются в удобные отдельные строки.
- **Добавлены обучающие карусели в Tools для WG Unlock и Ярлыков питания**: в обоих инструментах появилась кнопка «Как это работает» с пошаговым объяснением, чтобы быстрее разобраться с запуском и результатами.
- **Кардинально обновлено верхнее меню приложения**: панель теперь значительно лучше адаптируется под большие и маленькие экраны.
- **Добавлено меню «Ещё» для узких экранов**: второстепенные действия автоматически переносятся в отдельное выпадающее меню, поэтому верхняя панель не перегружается.
- **Улучшена читаемость на широких мониторах**: в быстрых действиях появились текстовые подписи рядом с иконками.
- **Исправлено перекрытие контента верхней панелью**: отступ основного экрана теперь подстраивается под фактическую высоту меню.
- **Улучшено меню «Ещё»**: теперь в выпадающем списке показываются только скрытые действия, а на маленьких экранах список корректно ограничивается по ширине и высоте.
- **Сделана более компактной правая часть верхнего меню**: быстрые действия теперь показываются только иконками без текстовых подписей.
- **Исправлен стиль кнопки «Обновить»**: она теперь визуально выглядит так же, как остальные кнопки панели.
- **Улучшены кнопки вкладок в верхнем меню**: вкладки получили более аккуратный сегментированный вид, ровные отступы и спокойное выделение активной вкладки.
- **Улучшены quick-actions в верхней панели**: кнопки быстрых действий получили единый glass-стиль, более спокойные состояния наведения/нажатия и лучше заметный фокус с клавиатуры.
- **Добавлена настройка отображения «Состояние сети» в верхнем меню**: теперь индикатор можно включать и выключать в разделе «Внешний вид» (по умолчанию выключен).
- **Улучшено меню трея и Dock на macOS**: пункты стали понятнее, а недоступные действия теперь корректно отключаются, чтобы в меню оставались только рабочие варианты.
- **Исправлено открытие приложения из трея**: при открытии окно теперь корректно восстанавливается из свернутого состояния и сразу получает фокус.
- **Сделано стабильнее обновление меню трея**: пункты меню теперь быстрее подхватывают изменения пути загрузки и статуса скачивания.
- **Исправлено повторное срабатывание события открытия настроек**: предупреждение о лишних слушателях больше не появляется при многократном открытии окна настроек.
- **Настройка одновременных загрузок стала понятнее**: теперь используется переключатель только между 1 и 2 задачами, а лимит синхронизирован с backend.
- **Полностью обновлён интерфейс меню «Настройки»**: разделы стали компактнее и проще для навигации в плотном системном стиле.
- **Добавлен сворачиваемый список разделов в «Настройках» для узких экранов**: на мобильных теперь удобнее переключаться между вкладками.
- **Содержимое разделов «Настроек» переработано в карточный формат**: параметры сгруппированы по смысловым блокам, поэтому их легче сканировать и настраивать.
- **Улучшена читаемость параметров в «Настройках»**: заголовки блоков и плотные отступы помогают быстрее находить нужные переключатели.
- **Сделан компактнее блок «Выбор качества загрузки» в настройках Downloader**: карточки выбора стали ниже и аккуратнее, поэтому раздел занимает меньше места по высоте.
- **Полностью переработан блок «Выбор качества загрузки» в настройках Downloader**: вместо двух больших карточек теперь используется компактный переключатель режимов с короткой контекстной подсказкой.
- **Во вкладке Tools инструмент File Sorter перенесён в раздел «Недоступно»**: карточка временно отображается как неактивная и не запускается из лаунчера.
- **В «Настройки -> Другое» добавлен раздел для разработчика**: после ввода секретного слова можно активировать скрытые функции во вкладке Tools.
- **Режим разработчика теперь сессионный**: после перезапуска приложения его нужно активировать заново.
- **В активном режиме разработчика кнопка меняется на «Отключить»**: можно быстро выключить доступ к скрытым функциям без перезапуска.
- **Обновлён фон вкладок приложения**: теперь все разделы используют единый общий фон, поэтому интерфейс выглядит целостнее.
- **Унифицированы шапки вкладок Tools, Backup и Randomizer под стиль Downloader**: верхний блок с иконкой, заголовком и подзаголовком теперь выглядит одинаково во всех основных вкладках.
- **Улучшен блок «Советы» во вкладке Backup**: теперь он расположен справа в шапке вкладки, а на маленьких экранах скрывается для более чистого интерфейса.

### Вкладка «Загрузчик»

- **Обновлён hero-блок вкладки «Загрузчик» в стиле Tools**: вверху теперь используется единый крупный заголовок с иконкой и подписью, как в вкладке инструментов.
- **Убраны «хлебные крошки» из «Загрузчика»**: верхняя часть вкладки стала чище и компактнее, без дополнительной навигационной цепочки.
- **Обновлена шкала прогресса в поле URL**: индикатор стал более аккуратным и теперь отображается тонкой линией внизу поля.
- **Снижен визуальный шум во время загрузки**: прогресс читается легче и меньше отвлекает от ввода ссылки.
- **Исправлено закрытие окна выбора качества по крестику**: теперь модальное окно стабильно закрывается по нажатию «×».
- **Исправлено подёргивание шкалы загрузки**: индикатор в поле URL больше не откатывается назад во время скачивания.
- **Исправлено переполнение шкалы в поле URL**: линия прогресса теперь всегда остаётся в границах поля и не выходит за правый край.
- **В окне «Выбор качества» добавлено скачивание превью**: теперь можно сохранить картинку видео прямо из модального окна.
- **Кнопка добавления в очередь стала компактнее**: вместо текста теперь используется понятная иконка.
- **Улучшено окно «Выбор качества»**: варианты теперь показывают структурированные параметры (разрешение, FPS, кодек, размер и контейнер), а блок выбора сразу отображает итоговый результат загрузки.
- **Сделано понятнее состояние загрузки форматов**: добавлены информативный статус с таймером, визуальные скелетоны ожидания и улучшенные состояния доступности для клавиатуры и экранных читателей.
- **Уточнено поведение окна «Выбор качества» при загрузке форматов**: до завершения поиска временно скрываются блок выбранного варианта и действия с источником/превью, а кнопка добавления в очередь остаётся недоступной.
- **Улучшен блок превью в окне «Выбор качества»**: добавлена строка с разрешением превью, кнопка копирования URL источника и понятное состояние, если картинка превью недоступна.
- **Добавлены подсказки горячих клавиш в окне «Выбор качества»**: теперь прямо внизу окна показываются клавиши навигации и действий, включая быстрый вариант «A» для добавления выбранного качества в очередь.
- **Переработаны кнопки в окне «Выбор качества»**: блок действий закреплён справа, визуальная кнопка «В очередь» убрана, а добавление в очередь теперь выполняется горячей клавишей `A`.
- **Исправлено определение разрешения превью в окне «Выбор качества»**: размер теперь корректно берётся и из доступных метаданных превью, поэтому строка с разрешением отображается стабильнее.
- **Сделан компактнее список вариантов в окне «Выбор качества»**: подробные бейджи параметров теперь свернуты по умолчанию, а в каждой карточке есть кнопка «Развернуть/Свернуть» для быстрого просмотра деталей.
- **Добавлена защита от повторных загрузок по режиму**: если ссылка уже есть в истории для того же режима (например, видео), она не добавляется повторно, но можно отдельно скачать аудио для той же ссылки.
- **Ускорена очередь загрузок в «Загрузчике»**: теперь можно скачивать до двух файлов одновременно, а оставшиеся задачи автоматически стартуют по мере освобождения слотов.
- **Улучшен интерфейс мультизагрузки в «Загрузчике»**: добавлены явные статусы активных задач, индикатор занятых слотов и более понятное управление отменой для параллельных загрузок.
- **Прокачана мультизагрузка в «Загрузчике»**: в активных задачах теперь виден прогресс по каждой ссылке, неуспешные загрузки можно быстро повторить, а лимит одновременных загрузок настраивается в Settings.

### Вкладка Tools

- **Улучшена стабильность File Sorter**: исправлена обработка ошибок при записи лог-файла, чтобы сортировка корректно показывала ошибку вместо аварийного завершения.
- **Улучшено поведение интерфейса File Sorter**: статус выбранной папки теперь обновляется корректнее.
- **Добавлено окно «Как это работает» для File Sorter**: теперь в инструменте есть пошаговая карусель с понятным объяснением работы сортировки.
- **Добавлено окно «Как это работает» для Проверки хеша**: в инструменте появилась пошаговая карусель с объяснением выбора алгоритма и сравнения результатов.

- **Добавлен новый инструмент File Sorter во вкладку Tools**: теперь можно выбрать папку и автоматически разложить файлы по категориям (изображения, видео, музыка, документы, архивы и прочее).
- **В File Sorter добавлен безопасный режим предпросмотра (dry-run)**: можно заранее увидеть результат сортировки без фактического перемещения файлов.
- **В File Sorter добавлен опциональный лог-файл**: при необходимости можно сохранить подробный список действий сортировки в отдельный файл.

- **Обновлена верхняя часть вкладки Tools**: заголовок с описанием и навигационная цепочка теперь объединены в один блок, а сама навигация расположена ниже описания.
- **Сделана понятнее навигация в Tools**: первый пункт хлебных крошек теперь называется «Tools» и использует иконку инструментов.
- **Полностью обновлён внешний вид вкладки Tools**: стартовый экран получил новый блок с описанием, навигационные хлебные крошки и более выразительные карточки инструментов в едином стиле.
- **Добавлена отдельная секция «Доступные инструменты» со счётчиком**: теперь сразу видно, сколько инструментов доступно на текущей системе.
- **Обновлён стиль внутренних экранов WG Unlock, проверки хеша и ярлыков питания**: карточки, отступы и акценты стали более аккуратными и лучше читаются на больших и маленьких экранах.

- **Переработано окно выбора качества во вкладке «Загрузчик»**: появились счётчики доступных форматов по вкладкам, блок с текущим выбором и быстрый выбор лучшего варианта для активной вкладки.
- **Упрощено окно выбора качества в «Загрузчике»**: убрана отдельная кнопка «Максимальное качество», чтобы не дублировать быстрый выбор в активной вкладке.
- **Улучшено управление выбором качества**: добавлена навигация клавишами ↑/↓ и запуск загрузки по Enter, чтобы быстрее выбирать формат без мыши.

- **Добавлена** вкладка **Tools**: быстрые действия в одном месте (WG Unlock, проверка хеша, системные инструменты).
- **Добавлена** проверка хеша файлов (MD5, SHA‑1, SHA‑256, SHA‑512) с сравнением ожидаемой суммы.
- **Добавлена** кнопка создания ярлыка перезагрузки и выключения Windows на рабочем столе.
- **Добавлен** блок «Конвертер форматов» как заготовка для следующего обновления.
- **Улучшены** кнопки, поля ввода и выпадающие списки во вкладке **Tools**: выше читаемость, понятнее состояния наведения/фокуса и аккуратнее визуальный стиль.
- **Сделано более предсказуемым управление клавишей Enter во вкладке Tools**: отправка WG теперь срабатывает только в форме WG Advanced и не мешает работе проверки хеша.
- **Улучшена доступность раскрывающейся панели Advanced**: кнопка теперь корректно сообщает состояние раскрытия, а фокус клавиатуры переходит в форму при открытии и возвращается на кнопку при закрытии.
- **Локализована справка по WG Unlock**: текст подсказки во вкладке Tools теперь отображается на выбранном языке интерфейса.
- **Скрыта карточка ярлыков питания на не‑Windows системах**: во вкладке Tools теперь показываются только действия, доступные на вашей платформе.

- **Уточнено поведение кнопки сетевых настроек**: она показывается только на платформах, где поддерживается прямое открытие системного окна.
- **Улучшена иерархия действий в WG Quick**: основное действие «Отправить» отделено от вторичных в блок «Доп. действия», чтобы быстрее ориентироваться в интерфейсе.
- **Освежён интерфейс вкладки Tools в более минималистичном стиле**: карточки и кнопки стали визуально спокойнее, с более чистыми отступами и меньшим визуальным шумом.
- **Убрана карточка-заглушка «Конвертер форматов (Скоро)»**: в сетке оставлены только реально доступные инструменты, чтобы рабочие действия находились быстрее.
- **Улучшено расположение блоков в Tools**: карточка WG Quick теперь выделена как основной сценарий, а вторичные действия сгруппированы компактнее.
- **Упрощён блок проверки хеша**: результат и состояния стали читать проще, с более аккуратной подачей статусов и занятости проверки.
- **Полностью обновлена навигация вкладки Tools**: теперь сначала открывается экран с крупными кнопками инструментов по центру.
- **Добавлен режим просмотра инструмента с кнопкой “Назад”**: в верхнем левом углу можно в один клик вернуться к экрану выбора инструментов.
- **Запоминается последний открытый инструмент**: при повторном открытии вкладки Tools интерфейс восстанавливает предыдущий инструмент, если он доступен на вашей платформе.

- **Улучшена проверка хеша в режиме двух файлов**: добавлена кнопка очистки второго файла, а ввод ожидаемого хеша теперь корректно сравнивается даже если в нём есть пробелы или переносы.
- **Уточнена подпись вычисленного хеша**: в результате проверки теперь показывается, какой алгоритм использовался для расчёта.
- **Сделан чище экран выбора инструментов в Tools**: убран дублирующий заголовок, улучшен контраст текста и добавлена более читаемая подложка под центральным блоком.
- **Добавлены короткие описания инструментов на стартовом экране Tools**: теперь проще понять назначение каждого инструмента до открытия.
- **Инструмент ярлыков Windows теперь виден на macOS**: можно заранее открыть раздел и увидеть доступные действия, даже если их создание остаётся доступным только в Windows.
- **Улучшена структура вкладки Tools**: внутренние модули вкладки переорганизованы, что упростит дальнейшие обновления и поддержку.
