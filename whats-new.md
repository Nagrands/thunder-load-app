# Что нового?

---

| Что изменилось? | Что это даёт? |
|---|---|
| Исправлено форматирование toast в настройках | Текст уведомлений снова отображается корректно, без видимых HTML-тегов |
| Блок «Дополнительно» в окне инструментов вынесен в отдельную секцию | Дополнительные действия и настройки теперь видны сразу |
| Удалена опция миграции инструментов из старого места | Интерфейс управления инструментами стал проще и понятнее |
| Исправлено отображение версии yt-dlp на macOS | В разделе «Инструменты» снова видно установленную версию yt-dlp |
| Обновлён File Sorter в разделе «Инструменты» | Теперь можно сначала посмотреть предпросмотр сортировки, увидеть категории файлов и только потом применить изменения |
| Расширены настройки File Sorter | Можно выбрать поведение при совпадении имен, включить подпапки и задать исключения для файлов и папок |
| Улучшен просмотр результата в File Sorter | В предпросмотре и итогах теперь есть поиск, фильтры по категориям и статусам, а результат можно экспортировать в файл |
| Расширен экспорт результата в File Sorter | Отфильтрованный список можно быстро скопировать в буфер обмена или сохранить в TXT, CSV и JSON |
| Улучшен список «Пропущены» в File Sorter | Теперь видно, какие именно файлы были пропущены и по какой причине: скрытый файл, исключение по расширению, исключение по папке или файл уже лежит в папке категории |
| Обновлён интерфейс File Sorter | Рабочая область, настройки и результаты стали визуально разделены, поэтому инструментом проще пользоваться на большом и маленьком экране |
| Обновлена «Первичная настройка» | Первый запуск стал пошаговым: теперь проще выбрать язык, нужные разделы и тему, а перед применением видно итог настроек |
| Внутренне переработан File Sorter | Инструмент стал лучше подготовлен к следующим улучшениям и будет стабильнее развиваться дальше |
| Унифицировано поведение всплывающих подсказок (tooltip) | Подсказки теперь работают одинаково во всех разделах приложения |
| Исправлены подсказки в истории загрузок | В списке записей снова корректно показываются все основные tooltip |
| Исправлено перекрытие подтверждений удаления в истории | Подсказки больше не накладываются поверх окна подтверждения |
| Улучшено действие «Повторить загрузку» в истории | После выбора экран плавно прокручивается к полю URL, а фокус ставится в него |
