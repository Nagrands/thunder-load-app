# Что нового?

---

| Что изменилось? | Что это даёт? |
|---|---|
| Упрощены настройки Backup | В настройках больше нет отдельного переключателя отключения Backup, остались только параметры отображения |
| Ускорен старт приложения | Основной экран быстрее доходит до состояния готовности, а второстепенные модули инициализируются позже |
| Упрощены Settings и Tools | Крупные модули разбиты на внутренние части, поэтому повторные открытия и переключения вкладок работают стабильнее |
| Перенесён Backup в Tools | Резервные копии теперь воспринимаются как инструмент внутри Tools, а не отдельная вкладка |
| Исправлена настройка окна после загрузки | Окно открытия файла после завершения загрузки теперь действительно включается и отключается этой настройкой |
| Исправлен текст в окне завершения загрузки | Сообщение больше не показывает HTML‑теги, а выглядит как нормальное окно с форматированием |
| Обновлены правила разработки | Больше ясности по архитектуре, сборке, UI и IPC для стабильных изменений |
| Обновлена документация | Единые и актуальные инструкции по разработке и скриптам |
| Исправлен счётчик шагов в первом запуске | Текст больше не показывает плейсхолдеры `{current}` и `{total}` |
| Улучшена кнопка отмены скачивания | Крестик выровнен по центру кнопки |
| Переработан экран настроек | Разделы стали визуально ровнее, статусы модулей читаются быстрее, а экран удобнее на узких окнах |
