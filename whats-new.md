### Вкладка Tools

- **Переработано окно выбора качества во вкладке «Загрузчик»**: появились счётчики доступных форматов по вкладкам, блок с текущим выбором и быстрый выбор лучшего варианта для активной вкладки.
- **Упрощено окно выбора качества в «Загрузчике»**: убрана отдельная кнопка «Максимальное качество», чтобы не дублировать быстрый выбор в активной вкладке.
- **Улучшено управление выбором качества**: добавлена навигация клавишами ↑/↓ и запуск загрузки по Enter, чтобы быстрее выбирать формат без мыши.

- **Добавлена** вкладка **Tools**: быстрые действия в одном месте (WG Unlock, проверка хеша, системные инструменты).
- **Добавлена** проверка хеша файлов (MD5, SHA‑1, SHA‑256, SHA‑512) с сравнением ожидаемой суммы.
- **Добавлена** кнопка создания ярлыка перезагрузки и выключения Windows на рабочем столе.
- **Добавлен** блок «Конвертер форматов» как заготовка для следующего обновления.
- **Улучшены** кнопки, поля ввода и выпадающие списки во вкладке **Tools**: выше читаемость, понятнее состояния наведения/фокуса и аккуратнее визуальный стиль.
- **Сделано более предсказуемым управление клавишей Enter во вкладке Tools**: отправка WG теперь срабатывает только в форме WG Advanced и не мешает работе проверки хеша.
- **Улучшена доступность раскрывающейся панели Advanced**: кнопка теперь корректно сообщает состояние раскрытия, а фокус клавиатуры переходит в форму при открытии и возвращается на кнопку при закрытии.
- **Локализована справка по WG Unlock**: текст подсказки во вкладке Tools теперь отображается на выбранном языке интерфейса.

- **Оптимизирована** работа интерфейса во вкладках **Tools** и **Randomizer**: уменьшена постоянная нагрузка на видеокарту, снижена лишняя анимация и более экономно обновляются визуальные индикаторы, поэтому приложение меньше нагревает систему при длительной работе.
- **Улучшена производительность вкладки Backup**: фоновые подсказки теперь приостанавливаются вне активной вкладки, список профилей рендерится легче при частой фильтрации, а анимации строк ограничены для больших списков.
- **Добавлена виртуализация списка профилей во вкладке Backup**: при больших страницах отображаются только видимые элементы, поэтому прокрутка и фильтрация работают заметно плавнее.
- **Добавлена виртуализация списка History**: на больших страницах теперь отрисовываются только видимые записи, за счёт чего прокрутка и работа с историей стали более плавными и с меньшей нагрузкой на систему.
- **Улучшена точность виртуализации History**: высота элементов теперь автоматически калибруется по фактическому размеру, поэтому прокрутка стала стабильнее при разной плотности и раскрытии деталей.
- **Оптимизированы подсказки по всему приложению**: тултипы теперь инициализируются лениво и переиспользуются без постоянного пересоздания при перерисовках, что уменьшает лишнюю нагрузку интерфейса.
- **Улучшено поведение всплывающих меню**: добавлен общий менеджер overlay, поэтому меню и контекстные панели закрываются централизованно и без лишних повторных обработчиков.
