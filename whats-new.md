<!-- version: 1.4.4 -->

# Release date: May 6, 2026

---

| Изменения                                                                                  | Что это даёт                                                                                                                                                                                                                                                  |
| ------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| «Проверка хеша» получила полноценный интерфейс проверки файлов                                   | Файл и параметры теперь разделены на две рабочие панели, алгоритм выбирается через фирменный dropdown, видны размер и готовность файла, а результат с копированием хеша обновляется без перезагрузки                                                           |
| Модальное окно выбора качества загрузки получило новый компактный дизайн                         | Форматы теперь читаются плотнее, карточка выбранного варианта заметнее, а действия с источником и превью вынесены в аккуратные иконки без изменения привычных сценариев скачивания и очереди                                                                   |
| Настройка отключения вкладки «Загрузчик» теперь скрыта без режима разработчика                   | Переключатель больше не показывается в обычном режиме, поэтому случайно скрыть основную вкладку нельзя, а dev-настройка остаётся доступной только после явного включения developer mode                                                                        |
| Раздел «О приложении» в настройках стал компактнее и полезнее                                    | Блок теперь показывает версии App/Electron/Chrome/Node в более плотной компоновке, а кнопки «Что нового», копирования сведений и проверки обновлений занимают меньше места и остаются доступны в одном месте                                                   |
| В модальном окне «Настройки» появился раздел «О приложении»                                      | В боковом меню настроек теперь есть отдельный пункт с краткой информацией о Thunder Load, версии приложения и версии Electron, поэтому базовые сведения о сборке доступны без поиска в footer или системном меню                                               |
| Полностью переработан фирменный знак приложения и tray-иконки для macOS и Windows                | Вместо старой weather-метафоры Thunder Load теперь использует одиночный знак `T`: основная иконка выглядит строже и чище в Dock и панели задач, а в трее macOS и Windows используется более лаконичная `T` внутри круга для лучшей читаемости на малом размере |
| Полностью удалено авто‑закрытие приложения                                                       | Настройка и таймер больше не появляются в разделе «Инструменты», приложение не завершится автоматически по старому отложенному сценарию, а legacy-параметры этой функции очищаются при запуске                                                                 |
| Для YouTube во вкладке «Загрузчик» добавлены video-preview и live preview                        | Под интерфейсом вкладки по-прежнему работает приглушённый фоновый preview, а по кнопке теперь можно открыть отдельное модальное окно с live player и звуком; при обновлении preview-source он старается продолжить и звук, и позицию того же ролика            |
| Статус инструментов Downloader перенесён в footer верхнего состояния                             | Блок с `yt-dlp`, `ffmpeg` и `Deno` теперь живёт рядом с обозначением текущего раздела в нижней панели, показывается только у верхней границы страницы и по-прежнему отключается из Настроек                                                                    |
| Навигация между вкладками теперь мигрирует из шапки в footer                                     | В верхней части экрана вкладки остаются в header, а при прокрутке вниз автоматически переезжают в нижнюю панель вместе с кнопкой быстрого возврата наверх                                                                                                      |
| В «Инструментах» обновлены системные быстрые ярлыки Windows                                      | Вместо «Диспетчера устройств» и «Параметров сети» теперь можно создать ярлыки «Установка и удаление программ» и «Чистка диска»                                                                                                                                 |
| «Проверка хеша» получила новый компактный интерфейс и исправленный Drag and Drop на Windows      | Область выбора файла и параметры проверки теперь занимают меньше места, появился быстрый сброс состояния, а перетаскивание файлов с Windows-путями и `file://` URI снова корректно подхватывается                                                              |
| File Sorter в русской локализации переименован в «Сортировщик» и получил более простой интерфейс | Предпросмотр, параметры сортировки, настройки и рабочая область теперь сгруппированы компактнее, поэтому рабочий сценарий читается быстрее и требует меньше лишних переходов                                                                                   |
