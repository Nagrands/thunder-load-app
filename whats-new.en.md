# Release date: June 3, 2026

---

| What changed                                                             | What you get                                                                                                                                                                                                                                                          |
| ------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Downloader now has a compact quality mode                                | Switch above the URL field to a compact view with preview, video and audio selectors, and Download starts the selected combination without opening an extra dialog                                                                                                    |
| The quality picker Audio section now includes MP3                        | You can choose an MP3 track directly in the quality dialog; the app downloads the best available audio stream and converts it through ffmpeg                                                                                                                          |
| The download queue can now be reordered by dragging                      | Grab a row by the dots on the left and drag it up or down to change the order of pending downloads without using separate move buttons                                                                                                                                |
| Quality selection now opens automatically after a recognized paste       | When a pasted link resolves with a preview and formats, the quality picker opens by itself; the behavior can be disabled in Downloader settings                                                                                                                       |
| Queue start now lets you choose the launch mode                          | When the queue has multiple items, Start asks whether to download only the first video or launch the whole queue; choosing the whole queue keeps the configured parallel download behavior                                                                            |
| The download quality dialog has a new compact design                     | Format choices are easier to scan, the selected option is clearer, and source/preview actions now live in clean icon buttons while the existing download and queue flows stay the same                                                                                |
| Added background and live preview for YouTube links in Downloader        | Pasting a YouTube URL still enables the muted decorative background preview, and now an explicit button can open a separate modal player with sound; when the preview source is refreshed, the player tries to continue the same video from roughly the same position |
| Downloader tools status moved into the footer top-state                  | The `yt-dlp`, `ffmpeg`, and `Deno` block now lives next to the current section label in the lower bar, appears only near the top of the page, and still respects the existing Settings toggle                                                                         |
| Section navigation now migrates from the header into the footer          | Tabs stay in the header near the top of the page, then move into the lower bar while scrolling down together with a quick back-to-top action                                                                                                                          |
| YouTube background preview in Downloader is more reliable                | Pasted links and retry actions now restore the muted video preview from the fast preview path and refresh it after cached metadata warms up                                                                                                                           |
| Compact quality picking is easier to read on Windows                     | Video and audio lists now open in a dark animated menu, long format labels no longer disappear on a white popup, and empty fields stay hidden until available formats load                                                                                            |
| Dropdowns now share one visual style                                      | Native selects, Settings custom menus, History filters, Tools controls, and Backup dropdowns now use the same dark treatment with consistent hover/focus states and readable Windows menu items                                                                        |
| Copied URL window behavior is more predictable                           | With the setting disabled, returning focus to the app no longer auto-pastes the copied link, and Windows now only focuses the app window without automatically maximizing it                                                                                          |
| Active downloads are protected from accidental reloads and closes        | The reload button and shortcuts are disabled while a download is running, and closing the window now shows a warning so the active job is not interrupted accidentally                                                                                                |
| Downloader optimization now has stricter safeguards                      | The queue no longer warms formats for simple preset jobs, preview cache invalidation is covered more tightly, and rapid URL replacements no longer render stale previews                                                                                              |
| Hash Check now has a compact redesign and fixed Drag and Drop on Windows | File selection and verification settings take less space, there is a quick reset action, and dropped Windows paths plus `file://` URIs are parsed correctly again                                                                                                     |
